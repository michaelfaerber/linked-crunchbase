Welcome to Linked CrunchBase!

A Linked Data API wrapping the REST API of CrunchBase, a database about technology companies and startups.

This service maps every request to Crunchbase. There also exist mappings of organizations and mappings of people in Crunchbase to DBpedia.

Authentication

To use our Linked Data API for Crunchbase, you have to authenticate to Crunchbase. To do so, enter your api key here (put the key in the username field and keep the password field empty). If you are not authenticated, only triples concerning the mapping to DBpedia are returned due to the licensing model of Crunchbase.

Example Linked Data API Usage

  • /api/company/facebook#id
  • /api/people/mark-zuckerberg#id
With curl:

curl -v -H "Accept:text/turtle" --header "Authorization: Basic {Base64-encoded key}" http://linked-crunchbase.org/api/organizations/facebook#id

Data Formats

Our Linked Data API supports JSON, JSON-LD, and RDF/N-Triples.

Crawling RDF Data with the Linked Data API

A detailed description of how to crawl an RDF data set based on our Linked Data API is given on GitHub. Due to licensing issues, we cannot provide a current RDF data set. An old data set from 2015 is provided online here and licensed partly under Creative Commons Attribution-NonCommercial License 4.0 (CC-BY-NC) and partly under Creative Commons Attribution License 4.0 (CC-BY).
